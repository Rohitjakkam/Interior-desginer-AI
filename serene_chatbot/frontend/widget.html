How to Embed This Chatbot on Your Website

  1. Start the backend server:
    cd serene_chatbot
    python -m uvicorn backend.main:app --host 0.0.0.0 --port 8001 --reload
  2. Add the chatbot to your website by including these lines before </body>:
    <!-- Serene Chatbot Widget -->
    <link rel="stylesheet" href="YOUR_PATH/chatbot.css">
    <script>
      window.SERENE_CHATBOT_CONFIG = {
        apiUrl: 'http://your-server:8001',
        companyName: 'Serene Design Studio',
        welcomeMessage: 'Need help with interior design?'
      };
    </script>
    <script src="YOUR_PATH/chatbot.js"></script>
  3. Or use the single embed script:
    <script src="YOUR_PATH/embed.js" data-api-url="http://your-server:8001"></script>
